Download & Submit Your Personal Data Sheet (PDS)

Please follow the instructions carefully. Only PDF files will be accepted for submission.

Step-by-Step Instructions

  1. Click the Download Excel Template button.
  2. Fill out your details carefully in the Excel file.
  3. Save a copy of the file for your records (optional).
  4. Convert the completed file to PDF format for submission.
How to Save as PDF (Microsoft Excel)
  1. Go to File > Save As or Export.
  2. Select your preferred folder to save the file.
  3. Under Save as type, choose PDF (*.pdf).
  4. Click Options... and select Entire workbook.
  5. Make sure All pages is selected.
  6. Click OK, then Save.
  7. After saving, upload your PDF file on the Job List page.
